# Small but Smart Calculator
 simple and user-friendly GUI Calculator built using Python. This project performs basic mathematical operations with error handling and a clean interface.

---

# Features
Addition
Subtraction
Multiplication
Division
Power (**)
Modulus (%)
Error handling (e.g., division by zero)

---

# Tech Used
Python
Tkinter (for GUI)

---

# How to Run
1. Install Python
2. Download or clone this repository
3. Run the file:
python Project1.py
